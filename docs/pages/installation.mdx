# Installation

You can install Datamint using npm:

```bash copy
npm install datamint
```

or using yarn:

```bash copy
yarn add datamint
```

## Prerequisites

Before you start, make sure you have the following software installed:

- [Node.js](https://nodejs.org/)
- [Docker](https://www.docker.com/)
- Datamint: You can install it using npm (`npm install datamint`) or yarn (`yarn add datamint`).

For more information on how to install and use these tools, you can visit their official documentation:

- [Node.js Documentation](https://nodejs.org/en/docs/)
- [Docker Documentation](https://docs.docker.com/)
